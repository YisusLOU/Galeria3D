DOM: Document Objecto Model

Métodos para interactuar con el DOM

  • QuerySelector(): devuelve el código delprimer HTML que tenga el elemento selector buscar
  • QuerySelectorAll(): devuelve en un arreglo del selector que tenga la valor buscar
  • getElementById: devuelve el codigoHTML del elemento con el nombre del id buscando, este metodo es mas rapido
  • createElements(): crea un nuevo elemento HTML
  • innerHTML: crea o modifica el contenido HTML del elemneto especificado
  • innerText: crea o modifica el contenido textual del elemneto especificado
  • appendchild(): agragar un nodo hijo al al final del elemento HTML especificado
  • removeChild(): elimina el nodo hijo indicado del elemento HTML especificado
  • insertBefore(): inserta un nuev nodo hijo antes del indicado como refencia en elelemento HTML especificado
